SISTEMA SEO JARVIS V1

FASE 0 — PREPARACIÓN Y AUDITORÍA

Objetivo:
Entender el estado actual de la web antes de tocar nada.

Verificar:

* Indexabilidad
* Arquitectura actual
* Meta tags
* Sitemap
* Robots.txt
* Schema
* Core Web Vitals
* Mobile usability
* Accesibilidad
* Imágenes
* SEO local
* Canibalización
* URLs

Entregable:

* Puntuación SEO estimada
* Lista priorizada de problemas
* Plan de implementación

---

FASE 1 — FUNDAMENTOS SEO

Objetivo:
Resolver el 80% de los problemas SEO con el menor esfuerzo.

Implementar:

* Titles únicos
* Meta descriptions
* Open Graph
* Twitter Cards
* Robots.txt
* Sitemap.xml
* Canonicals
* LocalBusiness Schema
* FAQ Schema
* Aggregate Rating Schema
* SEO Head por página

No modificar diseño.

Resultado esperado:

SEO Score:
60 → 80+

---

FASE 2 — INDEXACIÓN Y CRAWLABILITY

Objetivo:

Implementar:

* Footer links
* Breadcrumbs
* Canonicals avanzados
* Optimización de navegación
* Accesibilidad SEO

SEO Score:
80 → 88+
