# Black Mustache Gents Salon — Website

Premium barbershop website for **Black Mustache Gents Salon** located in JVC, Dubai.

## Overview

A complete multi-page React + Vite website with:
- 7 pages: Home, Services, Book Appointment, About, Gallery, FAQ, Contact
- Bilingual support (English / Spanish) via context
- WhatsApp booking integration
- Full SEO (meta tags, Open Graph, LocalBusiness schema, FAQ schema, robots.txt, sitemap.xml)
- Dark luxury aesthetic with gold accents

## Tech Stack

- **Framework:** React 18 + Vite 5
- **Routing:** React Router v6
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **SEO:** react-helmet-async

## Dev Server

Runs on port 5000 with host 0.0.0.0 (Replit proxy compatible).
